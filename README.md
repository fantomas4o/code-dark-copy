# Code Dark Copy

**Dark, elegant code blocks with a single, beautiful copy button.**

No duplicates. No conflicts. Works with **any theme** (Kadence, Astra, GeneratePress, etc.).

---

## Features

- **Monokai Sublime** dark theme  
- **Clipboard Копирай** → **Копирано!**  
- Auto syntax highlighting (Highlight.js)  
- Removes **all old copy buttons**  
- Full Gutenberg + Classic Editor support  
- Supports: `bash`, `yaml`, `php`, `js`, `docker`, `json`, etc.  

---

## Installation

1. Download code-dark-copy.php
2. Go to **WordPress → Plugins → Add New → Upload**  
3. Activate  
4. Done!
